## 📋 Что нужно сделать ТЕБЕ:

### 1. Узнай свой Telegram ID (2 минуты)
- Открой @userinfobot в Telegram
- Нажми Start
- Скопируй число (это твой ID)
- **ВАЖНО:** Этот ID нужен чтобы только ты мог управлять ботом!

### 2. Зарегистрируйся на GitHub (5 минут)
- Зайди на github.com
- Sign Up (регистрация)
- Подтверди email

### 3. Зарегистрируйся на Render (3 минуты)
- Зайди на render.com
- Sign Up through GitHub (регистрация через GitHub)

### 4. Создай репозиторий на GitHub (2 минуты)
- New repository
- Название: youth-feedback-bot
- Public (публичный)
- Create repository

### 5. Загрузи файлы на GitHub (3 минуты)
Из папки youth_feedback_bot загрузи все файлы:
- bot.py
- database.py
- config.py
- requirements.txt
- Procfile
- README.md
- .gitignore
- DEPLOY_INSTRUCTIONS.md

### 6. Подключи Render к GitHub (5 минут)
- На Render: New → Background Worker
- Выбери свой репозиторий youth-feedback-bot
- Runtime: Python 3
- Build command: `pip install -r requirements.txt`
- Start command: `python bot.py`

### 7. Добавь переменные окружения (2 минуты)
В настройках Render добавь:
1. BOT_TOKEN = ***REVOKED_TOKEN***
2. ADMIN_ID = [ТВОЙ ID ИЗ ШАГА 1]

### 8. Запусти деплой (2 минуты)
- Нажми "Create Background Worker"
- Жди 2-5 минут
- В логах должно появиться "Bot started!"

### 9. Протестируй бота (5 минут)
- Открой Telegram
- Найди @KBC_Youth_Feedback_Bot
- Напиши /start
- Используй /pending чтобы одобрить себя
- Протестируй /start_survey

## 📚 Полезные файлы:

- **DEPLOY_INSTRUCTIONS.md** - подробная инструкция с картинками что и как делать
- **README.md** - описание бота, команды, возможности
- **bot.py** - основной код (не трогай если не разбираешься в Python)

## ⚠️ Важные замечания:

1. **Токен бота** - это как пароль, не публикуй его нигде кроме Render
2. **ADMIN_ID** - обязательно укажи свой правильный ID, иначе не сможешь управлять ботом
3. **Бесплатный план Render** - бот может "засыпать" после 15 минут бездействия
4. **База данных** - на бесплатном плане сбрасывается при каждом обновлении

## 🆘 Если что-то не работает:

1. Проверь логи на Render (вкладка Logs)
2. Проверь что все переменные окружения правильно указаны
3. Проверь что бот запущен (статус Running)
4. Попробуй написать боту /start еще раз
